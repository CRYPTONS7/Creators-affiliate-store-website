# Team Task Manager

A modern full-stack collaboration platform where teams can create projects, manage members, assign tasks, and track progress efficiently.

## Features

- JWT Authentication (Login/Signup)
- Role-Based Access Control (Admin/Member)
- Project Creation & Team Management
- Add/Remove Project Members
- Multi-Member Task Assignment
- Dynamic Project-Based Member Selection
- Task Status Tracking
- Dashboard Analytics
- MongoDB Database Integration
- REST APIs with Next.js
- Railway Deployment Ready
- Responsive Modern UI

## Tech Stack

- Next.js 15
- TypeScript
- MongoDB + Mongoose
- Tailwind CSS
- JWT Authentication
- Railway Deployment
